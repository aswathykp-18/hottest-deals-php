🖥️ PLATFORM-SPECIFIC INSTRUCTIONS:

Windows (XAMPP):
   1. Download XAMPP from apachefriends.org
   2. Install and start Apache + MySQL
   3. Extract files to: C:\xampp\htdocs\hottest-deals\
   4. Open: http://localhost/phpmyadmin
   5. Create database: hrms
   6. Import: hottest_deals_full.sql
   7. Visit: http://localhost/hottest-deals/

Mac (MAMP):
   1. Download MAMP from mamp.info
   2. Install and start servers
   3. Extract to: /Applications/MAMP/htdocs/hottest-deals/
   4. Open: http://localhost:8888/MAMP/
   5. phpMyAdmin → Create database: hrms
   6. Import: hottest_deals_full.sql
   7. Visit: http://localhost:8888/hottest-deals/

Linux (Native):
   1. Install: sudo apt install php8.2 php8.2-mysqli mariadb-server
   2. Extract files to any directory
   3. Import: mysql -u root -p hrms < hottest_deals_full.sql
   4. Run: php -S localhost:8080
   5. Visit: http://localhost:8080/
